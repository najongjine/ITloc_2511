C언어가 너무 파격적이어서, 모든 언어는 C언어에서 영감을 받았어요

java, c, c++, c# 얘네들은 변수를 선언한때 타입을 먼저 명시해요

int a;
char c;


하지만 typescript 는 변수를 선언할때

let 라는 키워드를 써요.
옛날 스타일은 var 를 써요. 요즘엔 var은 안써요.

let a : number = 1;

타입스크립트는 타입을 변수이름 오른쪽에 써요.

언어와 라이브러리 이런것들은 박사님들이 만들어요

그러면 그 박사님이 만든 규칙대로 가야해요.

예를들면 자바는 존 고슬링 이라는 박사님이 만들었어요


타입스크립트는 마이크로소프트에서 만들었고,
**아네르스 하일스베르(Anders Hejlsberg)** 라는 박사님이 참여했어요

타입스크립트를 쓸때는 마이크로소프트의 규칙을 따라야해요

파이썬을 쓸때는 파이썬 규칙을 따라야 해요

그리고 미리보기를 말씀 드리면,

인공지능, 게임, 메타버스, 3D 이쪽은 표준이 없어요.

다행이도, 웹 앱 이쪽은 표준이 있어요.
쉬운말로 웹 앱 쪽은 복사붙이기로 거의 만들어요


모든 프로그래밍 언어는

변수 선언을 하고,
변수는 메모리의 공간을 할당 받고,
우리는 변수 이름으로 메모리 공간에 접근한다.
